# Laravel Sail

https://laravel.com/docs/10.x/sail

## Requirements

- PHP v8.2
- Composer v2.5

## Installation

- Install Docker Desktop
- `composer create-project laravel/laravel laravel-sail`
- `composer require laravel/sail --dev`
- `php artisan sail:install`
  - 0 = MySQL
- `./vendor/bin/sail up`
